# Resumen
Este proyecto consiste en una aplicación de escritorio básica desarrollada en Java utilizando JavaFX. El objetivo central del ejercicio es aprender a maquetar interfaces de usuario estructuradas en forma de tabla o matriz bidimensional utilizando el contenedor dinámico GridPane.

## Características
Estructura Matricial (GridPane): Distribuye los componentes en un sistema de coordenadas de filas y columnas, permitiendo un control milimétrico sobre la posición de cada control.

Control de Espaciados Fijos: Define distancias de separación uniformes tanto en sentido horizontal (setHgap) como vertical (setVgap) para evitar que los componentes visuales queden pegados entre sí.

Imports Limpios Residuales: Aunque en la cabecera del archivo se importan layouts alternativos como HBox y VBox, el código se enfoca y resuelve exclusivamente con la API de rejilla.
